# Luciano Iacobelli — Site Images Backup

This folder is for storing downloaded copies of all images used on the site.
All images are currently hosted on the old Shopify CDN (`cdn/shop/` at lucianoiacobelli.com).

## How to download all images

Run the PowerShell script `download-images.ps1` in this folder. It will download
every image into subfolders by category.
